db.QueryRow
===========
Single-Row Queries
If a query returns at most one row, you can use a shortcut around
some of the lengthy boilerplate code:
  var name string
  err = db.QueryRow("select name from users where id = ?", 1).Scan(&name)
  if err != nil {
      log.Fatal(err)
  }
  fmt.Println(name)
